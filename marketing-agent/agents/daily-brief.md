# Agent: Daily Marketing Brief

## Role
Run every morning. Produce a prioritized action list for the day's marketing activities.

## Brief Structure

```
# Fun Small Talk - Daily Marketing Brief
Date: [date]

## Top Opportunities Today
[3-5 specific posts/threads to engage with — with links and suggested approach]

## Content to Post
[1-2 pieces of content ready to post from the drafts folder, or to create today]

## DM Follow-ups
[Any DM sequences that need a follow-up message today]

## Weekly Progress
- Comments drafted this week: X
- Quora answers drafted: X
- Original posts created: X
- DM sequences active: X

## This Week's Focus
[What platform / content type to prioritize based on what's working]
```

## How to Run

1. Search for new Reddit/Quora posts in target communities (last 24 hours)
2. Check for any new Instagram/TikTok engagement that needs a DM response
3. Pull any pending drafts from `outputs/drafts/` that are ready to post
4. Generate 3-5 fresh comment drafts for the best opportunities found
5. Output the full brief

## Output
Save to: `outputs/daily-brief-[YYYY-MM-DD].md`
Print summary to screen.
